# To-Do

A simple to-do list web app. Add tasks, check them off, filter by status, and clear completed items. Tasks persist in the browser via `localStorage`.

No build step or dependencies — just static HTML/CSS/JS.

## Run locally

Open `index.html` in your browser, or serve the folder with any static file server.

## Files

- `index.html` — page structure
- `style.css` — styling
- `app.js` — app logic (add/toggle/delete/filter, localStorage persistence)
